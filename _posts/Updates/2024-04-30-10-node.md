- **Docker Hub**: [artifriends/saseul-network](https://hub.docker.com/r/artifriends/saseul-network)

### v2.2.0.3 (2026. 02. 23)

- Reduced network traffic by 50–80% between updated nodes
- Added transaction query API
- Added inspect API for contract-bound status data
- Added sendtransaction support via wallet and script
- Fixed permission errors occurring in certain situations
- Minor bug fixes

### v2.2.0.2 (2026. 02. 16)

- Minor bug fixes

### v2.2.0.1 (2026. 02. 13)

- Added backward compatibility layer for legacy block data
- Added experimental GPU miner
- Improved sync performance and fork selection stability

### v2.1.9.6 (2023. 07. 20)

- Hard fork was proceed at 625,000 resource block
- If the update has not been carried out since v2.1.7.6, there might be a problem with synchronization
- Mining difficulty problem has been fixed

### v2.1.7.6 (2023. 04. 02)

- Added some script
- The issue of excessive network traffic has been fixed
- Now you can set ports other than port 80
- Fixed synchronizing issues, stabilization issues
- Check your own node status using terminal and send transaction or register smart contract

### v2.1.6.2 (2022. 10. 19)

- Fixed synchronizing issues, stabilization issues

### v2.1.4.3 (2022. 09. 25)

- Now supports arm64

### v2.1.3 (2022. 09. 12)

- Bug fixes from v2.1.2 (not critical)
- If you were using v2.0.x, you should proceed with the v2.1.2 patching method first

### v2.1.2 (2022. 09. 09)

- SASEUL v2.1.2 is automatically forked at main block number 150,000. It is no longer compatible with v2.0.x
- Token issuance function will be added after the fork completes

### v2.1.0 (2022. 08. 24)

- Modified to operate stably even in low-spec nodes
- Unstable release — hard fork planned after full testing

### v2.0.8 (2022. 07. 18)

- Node stabilization

### v2.0.7 Hotfix (2022. 07. 15)

- Fixed mining problem caused by intermittent synchronization stopping

### v2.0.7 (2022. 07. 14)

- Fixed synchronization stopping if the internet is temporarily disconnected

### v2.0.6 (2022. 07. 08)

- ForceSync on specific hosts is now available

### v2.0.4 (2022. 06. 30)
